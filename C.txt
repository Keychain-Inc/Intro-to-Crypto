Crypto101

Cryptocurrency is a technology which allows for cryptographically (complex algorithms) secure transactions using a currency or asset which can be transfered using a digital network. Cryptocurrencies are such assets that are able to be easily send, receive and use on what is usually a blockchain. Crypto as a movement is not just about the technology which can change how we act and think about money, but a n idealogy based on self-sovereignty, decentralization, autonomity, permissionllessness and mpst importantly openness.
